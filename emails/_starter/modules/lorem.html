How to Create an Email

  1. Copy the '_starter' directory and rename it for your email.

  2. In 'layouts.html', add styles in the 'head' block tag.

  3. For each module:

    1. Create a new file in the modules folder. A placeholder file is included if you want to copy that.
    2. Extend the module template you'd like to use. Available module templates are found in the 'templates/modules' folder.
    3. Fill out the block tags of the module with your content.
    4. In 'layout.html', include the module in the 'content' block tag. You can remove the include for the placeholder file.

For more information, visit the Twig documentation on design templates.
